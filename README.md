Modification and optimization of a legacy Excel file used in a retail business to monitor and control part of its economic performance.

---

## Project Goal

Improve an existing internal Excel tool to provide a clearer and more structured visualization of key business metrics, enabling decision-makers to act more efficiently.

This project is **not focused on data analysis itself**, but on:

* Enhancing usability
* Improving data visualization
* Structuring information for better decision-making

---

## Key Features

* 📊 **Clear visual dashboards** for quick interpretation
* 🧩 **Structured data model** separating raw data and visual layers
* 🔁 **Template-based design** for reuse and scalability
* 🧭 **Index sheet** for easy navigation across the workbook
* 🧪 **Example sheet** with fully anonymized and generalized data

---

## Data Source

Raw data is not manually introduced in this file.

It is automatically generated from external sources (CSV files exported from another platform).
This integration is not included in the current version of the repository but may be added in the future.

---

## Data Privacy

All data included in this repository has been:

* Fully **anonymized**
* **Generalized** (e.g., "Empresa A", "Producto 1", etc.)
* Modified to prevent any real business information leakage

This ensures the **confidentiality and integrity** of the original company.

---

## File Structure (Excel)

The workbook is organized into:

* **Index sheet** → navigation across all sections
* **Template sheets** → reusable structure for analysis
* **Raw data sheets** → base data for calculations and visuals
* **Visualization sheets** → dashboards and summaries
* **Example sheets** → filled with anonymized data

---

## Use Case

Designed for business environments where:

* Data is generated externally (ERP / platform exports)
* Teams need **quick insights without technical complexity**
* Decision-makers require **clear and actionable visuals**
